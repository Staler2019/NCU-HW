# Linux OS & Windows OS

不同 OS 會有不同的內置 library 來幫助作業系統運行，當然，我們如果要使用這些 OS 的服務，也就是運行程式的話，就需要依賴這些 library。Windows 的服務是建立在 Win32 api 上，而 Linux 的服務是建立在 system calls 上，雖然希望要做的工作一樣，但是不同平台就會因此而跑不起來。

## The Answer is `NO`
