# Scanner

把檔案中的文字分為一塊一塊有意義的最小區塊(split into `tokens`)， tokens 有了各自的意義，才能進行之後的語法檢查、語意檢查、及 AST 的建立等，例如下面是一個 python code，主要由 3 個部分組成:

` a = 10.5`

- 變數(var): a
- 指定運算符(operator): =
- 值、型態: 10.5 is a float
